Setup Steps
  1. Identify Devices

    List all Wi-Fi connected devices: laptops, phones, TVs, tablets, IoT gadgets.

  2. Find Router IP

    Run ipconfig on Windows or check Wi-Fi settings → Gateway on Android/Mac.

  3. Draw Network Diagram

    Place router at center, connect all devices in a star pattern, add internet cloud at top.

  4. Verify & Test

    Type router IP in browser — login page confirms correct IP. Test internet on all devices.

Validation Checklist
All devices connected to Wi-Fi
Router IP found (e.g. 192.168.1.1)
Router login page accessible via browser
All devices connect to same router
Internet works on all devices
Diagram shows star topology
Router connects to internet cloud
Each device has unique IP address
